Set and Map
Set

A set is well-defined a collection of distinct objects. The object is called the set's elements or members. \begin{align} x \in A \end{align} means that "x is an element of A". "x is a member of A", "x belongs to A", "x is in A", and "x lies in A" are the same. Another expressions "A includes x" and "A contains x". \begin{align} A \ni x \end{align} is another notation.

The negation of "\( x \in A \)" is denoted

\begin{align} A \not\in x \end{align}
