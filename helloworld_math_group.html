Group Theory

Group

A group is a set that have algebratic structure. The elements of the set equip with an operation that combines any two elements to form a third element.

Def: (Group Definition)

\begin{align} G:\ A \ Set \nonumber \end{align} \begin{align} *:G \times G \to G :\ A\ Map \nonumber \end{align} \begin{align} (G,*)\ is\ a\ group\ \ \overset{def}{\Longleftrightarrow} \end{align} \begin{align} i)\forall a,b,c \in G,\ a(bc)\ =\ (ab)c \end{align} \begin{align} ii)\exists e \in G\ \ s.t.\ ae=ea=a \end{align} \begin{align} iii)\forall a \in G,\ \exists b \in G \ \ s.t.\ ab=ba=e \end{align}

For example, ( \( \mathbb{R},\ +\) ),( \( \mathbb{R},\ *\) )
